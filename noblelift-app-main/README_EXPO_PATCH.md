
# Noblelift — Expo/React Native UI Patch (2025‑09‑13)

**Что это**
Готовые экраны/сервисы под **React Native + Expo** (без Vite). Подменяют моковые списки на реальные API, чинят карточки задач, «Готово», «Доступные», «Создать», «Профиль», «Автопарк». Минимальные стили и навигация через `@react-navigation` с табами.

**Как применить**
1. Скопируйте папки `src/` и `App.tsx` в ваш проект Expo (или перенесите файлы по месту).
2. Убедитесь, что установлены навигационные пакеты (если нет):
   ```sh
   npx expo install @react-navigation/native @react-navigation/native-stack @react-navigation/bottom-tabs react-native-screens react-native-safe-area-context
   ```
3. В `src/config.ts` укажите базовый URL API (по умолчанию `"/api"`). Токен ищется в `AsyncStorage` по ключу `token`.
4. Запустите: `npx expo start`.

**Экранные вкладки**
- Задачи — назначенные мне + безназначенные, фильтры, карточки полные, «Готово» работает.
- Доступные — только без `assigneeId`.
- Создать — поля из ТЗ: `type:"common"` всегда, при «Личной» — `isPrivate:true`, `assigneeId = me.userId`.
- Профиль — /profiles/me.
- Автопарк — полные карточки, форма добавления сверху (видна супер‑админу), кнопка «Взять».

Если у вас другой роутинг — просто возьмите файлы из `src/screens` и подключите в свой навигатор.
